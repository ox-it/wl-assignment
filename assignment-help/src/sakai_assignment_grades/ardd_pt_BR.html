Retornar uma Atividade para um Aluno

Observação: Para concluir esses procedimentos, você deve possuir um papel com as permissões necessárias. Para determinar seu papel, siga as instruções em Papéis dos Participantes. Para um entendimento básico sobre papéis e permissões, veja Permissões e papéis: Resumo.

Quando você clica em Atividades no menubar, você vê uma lista de atividades. A coluna "Em / Novo" exibe o número de submissões dos alunos (o valor "Em" ) e o número de novas submissões (o valor "Novo") para cada atividade. Esses valores permitem que você saiba se você precisa realizar uma ação em relação a uma atividade.

Instruções detalhadas para avaliação de atividades, incluindo informações retorno de uma atividade, estão no documento Avaliar Atividade ou Rever Situação do Aluno. Note que depois de ter começado a avaliação da atividade de um aluno, você pode retornar a atividade para o aluno com ou sem nota. Você pode fazer comentários, tanto dentro do texto submetido pelo aluno e dentro da caixa de texto Comentários do Instruto. Para permitir que o aluno reenvie a atividade, marque Permitir Reenvio. Quando você estiver pronto para retornar a atividade, clique em Returnar para o aluno na parte inferior da tela. Em "Submissões" na lista para a atividade, sob o título "Situação", você verá o valor "Devolvido" para o aluno em questão.
