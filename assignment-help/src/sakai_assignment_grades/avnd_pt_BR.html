Link da Atividade no Quadro de Notas

Em Atiivdades, instrutores tem duas opções para criar um relacionamento inicial entre uma atividade e um item de Quadro de Notas:

  • Selecionar Adicionar Atividade para o Quadro de Notas.
  • Selecionar Associar com entrada de um Quadro de Notas existente.

Em geral, a escolha Adicionar Atividade para o Quadro de Notas é a opção mais simples porque você pode inserir e editar notas de atividade de um lugar (na ferramenta Atividades), e eles vão exibir tanto lá quanto no Quadro de Notas.

Estas recomendações são baseadas nas seguintes questões sobre a relação entre as ferramentas Atividades e Quadro de Notas:

Usar a opção Adicionar Atividade para o Quadro de Notas

  • Você deve usar a ferramenta Atividades quando você quiser editar o valor do ponto e data de conclusão de uma atividade. Além disso, você necessita da nota da atividade na ferramenta Atividades.

  • As configurações do Quadro de Notas para este item são Liberar este item para alunos, Inclua este item no cálculo de nota de curso , e Categoria (se você criou categorias).

  • Uma nota que você atribui para uma atividade individual através da ferramenta Atividades não pode ser substituída no Quadro de Notas.

  • Notas que você atribui para uma atividade individual não são exibidas no Quadro de Notas (mesmo para o professor) até você liberar as notas ou retornar uma nota a um aluno a partir da ferramenta Atividades. No Quadro de Notas, as notas atividade não será exibida para os alunos se você não tiver selecionado (no Quadro de Notas) a configuração Liberar este item para os alunos.

  • Pode ser confuso que, depois de começar com Adicionar Atividade no Quadro de Notas, você precise usar o padrão Associar com uma entrada do Quadro de Notas existente definindo a edição na ferramenta Atividades.

Observação: As limitações do uso desta opção são que você não pode usar o Quadro de Notas para substituir uma nota de uma atividade, e você deve fazer alterações nas configurações, bem como quaisquer alterações de nota, na ferramenta Atividades.

Usar a opção de Associar com uma entrada do Quadro de Notas existente

  • Você deve alterar o valor do ponto e as configurações de data de conclusão na ferramenta Atividades e Quadro de Notas (ou seja, você deve manualmente sincronizá-los).

  • Notas atribuídas na ferramenta Atividades não são exibidas no Quadro de Notas (mesmo para o professor) até que você libere as notas ou retornar uma nota a um aluno a partir da ferramenta Atividades. No Quadro de Notas, as notas da atividade não será exibida para os alunos se você não tiver selecionado (no Quadro de Notas) Liberar esse item para os Alunos.

  • Uma nota atribuída para uma atividade individual através da ferramenta de Atividades podem ser substituídas no Quadro de Notas. Esta alteração será refletida na ferramenta Atividades.

  • Se você clicar em Liberar Notas na ferramenta Atividades, qualquer nota que foi alterada na ferramenta Quadro de Notas será substituída na ferramenta Atividades.

Observação: A maior desvantagem do uso dessa opção é que se você quiser editar as configurações, você deve fazer as alterações nas ferramentas Atividades e Quadro de Notas. A vantagem é que você pode usar o Quadro de Notas para substituir uma nota de uma atividade.

Informações adicionais

  • Quando você importar as atividades e itens do Quadro de Notas de outro site, os itens no Quadro de Notas que você criou com a opção Adicionar Atividade ao Quadro de Notas não aparecerá inicialmente no Quadro de Notas. No entanto, serão adicionadas quando você editar e salvar cada atividade importada.

  • Para importar atividades que você criou usando a opção Associar com entrado existente no Quadro de Notas, você terá que usar a ferramenta Atividades para reassociá-los com os itens apropriados do Quadro de Notas. Atividades importadas terão Adicionar Atividade para o Quadro de Notas selecionado por padrão; para associar o item com a entrada do Quadro de Notas existente, selecione Associar com entrada existente no Quadro de Notas e escolha a entrada Quadro de Notas desejada antes de postar a atividade.
