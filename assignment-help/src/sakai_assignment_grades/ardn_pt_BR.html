Atividades: Resumo

O que faz

Para os cursos, a ferramenta Atividades permite aos Instrutores criar, distribuir, recolher e dar notas a atividades on-line. Atividades são privadas; submissões dos alunos não são visíveis para outros usuários do site.

Principais Conceitos

Várias opções de avaliação: Você pode escolher notas de letras, pontos, aprovado / reprovado ou atividades sem avaliação.

Várias submissões: Você pode optar por permitir que os alunos submetam uma atividade várias vezes.

Retornando uma atividade: Atividades podem ser devolvidas, com ou sem notas, para reenvio.

Revisão offline: Os instrutores podem baixar todas as submissões de uma atividade para o seu computador ao mesmo tempo.

Acesso a nota: Quando os instrutores liberam as notas de uma atividade, os alunos podem acessar os comentários do instrutor e as suas notas.

Ver como aluno: Clicando em Ver como aluno permite que você, como instrutor ou responsável pelo site, visualize as atividades como elas apareceriam para os seus alunos ou participantes do projeto. Para saber mais sobre o uso, veja Ver Atividade Como Aluno.

Compromisso: Esta opção exige que os alunos marquem uma caixa para afirmar que eles não têm dado ou recebido ajuda em uma atividade antes deles submete-la.

Fornecer feedback imediato ou programado: Os instrutores podem definir um modelo de resposta para fornecer feedback imediato para uma atividade submetida, ou para fornecer feedback em uma data e hora.

Notas privadas:Instrutores podem adicionar notas privada (ou seja, os alunos não podem acessar as notas) a uma atividade.

Considerações

  • A ferramenta Atividades pode enviar notas para o Gradebook. Para saber mais sobre a relação entre as ferramentas Atividades e Quadro de Notas, veja Em ferramentas Atividades, qual método devo usar para vincular minha atividade para o Quadro de Notas?
